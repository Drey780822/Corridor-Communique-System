# 🏢 Corridor Hill Communiqué System

A web-based communication platform built for student residence management at Corridor Hill (eMalahleni).
The system enables administrators to share important updates while allowing students to stay informed and engaged.

---

## 🚀 Overview

The **Corridor Hill Communiqué System** is designed to improve communication between residence management and students.

It provides a centralized platform where:

* Admins can post announcements, events, and maintenance updates
* Students can view updates and stay informed
* Notifications are sent via SMS for urgent communication

---

## ✨ Features

### 👨‍💼 Admin Features

* Create and manage announcements
* Post events and important updates
* Send SMS notifications to students
* Manage student access

### 🎓 Student Features

* Secure login (students only access)
* View announcements and updates
* Receive SMS notifications
* Stay updated with residence activities

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** PHP
* **Database:** MySQL
* **SMS Integration:** Twilio

---

## 📱 SMS Notification System

The system integrates with Twilio to send real-time SMS notifications to students, ensuring important updates are never missed.

---

## 🔐 Access Control

Only registered students are allowed access to the system, ensuring privacy and controlled communication within the residence.
